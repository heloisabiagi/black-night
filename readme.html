Black Night - Documentation

Thanks for purchasing Black Night. This document will provide usage instructions so you can enjoy Black Night theme at its best. If you find any errors or bugs, please contact us here.

1. Starting point - Installing, Files and Folders

Once you've downloaded and unzipped the theme, you'll find a bunch of files and folders. The main files of the theme are demo-original.html, demo-alternate.html, which contain the source code for the theme's pages and can be customized with your own data. The assets folder contains the styles, scripts, fonts, images and other sources referenced in the html files.

The only files/folders needed to be put in production for the theme to work properly are the html file that contains your sites's data and the 'assets' folder. All the other files and folders are settings files with no need to be put in production (unless you intend to use 'Grunt.js' or SASS compilers in production environment), and will be explored later in this tutorial.

If you intend to use 'Grunt.js', you have to install the modules listed in the package.json file. To command line and 'npm' users, it's a pretty simple proccess:

npm i

A new folder, node_modules, will be installed in the theme's directory.

Next, let's have a look at the assets and source folders in order to understand how to make a better use of the theme's funcionalities.

1.1 The 'assets' folder: files, images and fonts

As mentioned before, the assets folder is the main folder which contains all the files called inside the html files. If you use 'Grunt.js' or 'SASS/Compass', this is also the final destination for the compiled files. Once you open the folder, you can find the following subfolders: 'fonts', 'images', 'inc', 'js' and 'styles'.

The fonts folder contains the icon fonts used in this theme. The theme fonts itself are Google fonts downloaded directly from the Google Fonts api. This theme supports two icon fonts libraries: 'Glyphicon', default to Bootstrap, and 'Font Awesome'.

The images folder contains the images used in this theme, which are split in 2 directories: 'bg' (for CSS background images) and 'body' (for images used in the html files). The 'original' folders contain original artwork for this theme, and the 'freeimages' contain images from 'Freeimages.com'. In order to use the images from Freeimages.com, you need to create an account and get your own copy of the desired images. The ones used in this layout are for demonstration purposes only.

The inc folder contains some special files, like the 'mailer.php' called through AJAX to send emails.

The js and style folders contain the Javascript and CSS files. Both folders have the original and the minified versions of the files, and it's the minified versions that are used in the theme. You can change the original files, however, if you wish to use them instead of the minified version, you have to replace the calls in the html files.

2. Theme features

2.1 Color schemes

Black Night has 5 built in color schemes ready to use: blue, green, orange, pink and purple. In order to change the theme color, you have to locate the default stylesheet (blue) in your html file:

<link rel="stylesheet" type="text/css" href="assets/style/style-blue.min.css">

And change it for the desired color scheme:

<link rel="stylesheet" type="text/css" href="assets/style/style-pink.min.css">

If you're using 'SASS/Compass', you can create infinite possibilities of color schemes for your site. All you need to do is to go to the source/sass folder, open any of the style-*.scss files, save it again with the desired color name and then, inside the file, replace the $main-color variable with your new desired color. The variations (darker and lighter) are automatically generated, but you still can set them manually if you wish by changing the $darker-main-color and $lighter-main-color variables with new values. Once you compile this brand new file (that will be generated at the assets folder, don't forget to replace the stylesheet code in your html file.

2.1 Plugins

In order to enhance user experience, Black Night makes use of the following Javascript plugins:

  • ScrollTo: makes scroll navigation easier.
  • Isotope: a fancy way to display and filter data. Used in the Portfolio section.
  • Magnific Popup: responsive lightbox plugin. Also used in the Portfolio section.
  • Owl Carousel: fancy responsive carousel. You can find it in action at the Testimonials section.
  • Wow: a plugin to reveal animations during scrolling.

The final compiled version of these libraries ('vendors.js' and 'vendors.min.js') can be found at the 'assets' folder, while the original individual files are located at the source/js/vendors directory.
